## 1. Reflection: Coding in Markdown

**Task:** My objective was to use GitHub Flavored Markdown to create a structured, web-ready document that includes professional formatting without using a complex word processor.

## 2. Reflection: Mock Interview Experience
**Situation:** The assessment required the inclusion of a mock interview video to showcase work readiness.

## 3. Reflection: Using GitHub Pages
**Situation:** The final requirement for the portfolio was to deploy the project as a live website.

**Task:** I had to configure GitHub Pages to host my repository's content so that it could be accessed via a public URL by EXAMINERS. 

**Action:** I accessed the repository settings and enabled GitHub Pages, selecting the main branch as the SOURCE. I verified that the README.md was being correctly parsed and that all links and the embedded video functioned in the live environment.

**Result:** The portfolio is now live and accessible. This experience taught me how to manage basic web hosting and deployment pipelines, which are critical skills in a modern IT environment.
